# Kepha: Shared Reading

Kepha is a social reading platform where people discover books, form clubs, and grow through shared ideas. This application is built with Next.js, Firebase, and uses Genkit for its AI-powered features.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:9002](http://localhost:9002) with your browser to see the result.

You can start by exploring the app, discovering new books, and joining or creating book clubs.

## Features

- **Book Clubs:** Create and join book clubs to read and discuss books with others.
- **AI Discussion Prompts:** Generate thought-provoking questions for chapter discussions.
- **AI Discussion Summaries:** Get a quick summary of what others are talking about.
- **Personal Notes:** Take private notes on chapters as you read.
- **Read Aloud:** Listen to an AI-generated audio version of any chapter.
- **Personalized Discovery:** Find your next favorite book with AI-powered recommendations.
